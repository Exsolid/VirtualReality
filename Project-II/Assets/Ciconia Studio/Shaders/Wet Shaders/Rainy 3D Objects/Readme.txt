Readme - Wet Shaders - Rainy 3D Objects
_______________________________________


HDRP: 
Support Unity versions
2019.4.x LTS
2020.3.x LTS

By default, the package is set up to be compatible for Unity 2019.4.x. If you are using Unity 2020.3.x delete the HDRP folder 
and unpack the HDRP-Rainy 3D Objects_2020.3.x.unitypackage.

Documentation link : https://docs.google.com/document/d/1aCE4aRP94IeXVO4R4afPavQGbAuwQdkt1zo5YfJMeaA/edit?usp=sharing
